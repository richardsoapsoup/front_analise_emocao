# Frontend - Guia de Execução

Este é o frontend do projeto. Siga os passos abaixo para rodar localmente.

## 📦 Pré-requisitos

- Node.js instalado (recomendado versão LTS)
- npm ou yarn

## 🚀 Como executar o projeto

1. **Abra o terminal**

2. **Acesse a pasta do projeto**
```bash
cd front_analise_emocao
```

3. **Instale as dependências**
```bash
npm install
```

4. **Execute o projeto**
```bash
npm run dev
```

5. **Acesse no navegador**
Normalmente estará disponível em:
```
http://localhost:3000
```
ou
```
http://localhost:5173
```

## 🛠️ Scripts disponíveis

- `npm run dev` — inicia o servidor de desenvolvimento
- `npm run build` — gera a build de produção
- `npm run preview` — visualiza a build de produção

## 📁 Estrutura básica

- `src/` — código fonte
- `public/` — arquivos estáticos
- `package.json` — dependências e scripts

## ⚠️ Observações

- Certifique-se de estar na pasta correta antes de rodar os comandos
- Caso haja variáveis de ambiente, configure um arquivo `.env`
